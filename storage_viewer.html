Storage Viewer - Duolingo Follow Dates

This page allows you to view and manage follow date data stored in the extension.

Automatic Synchronization & Data Preservation

The extension now automatically synchronizes storage with Duolingo while preserving historical data:

  • When you open the extension, it checks if there are people you stopped following directly on Duolingo (without using the extension) and marks them as "inactive" in storage instead of removing them.
  • The same happens with people who stopped following you.
  • This preserves your complete social history while keeping track of current relationships.
  • You can use the filter options below to view active users, inactive users, or all users.

Backup and Data Transfer

You can export your data to a JSON file and import it later. This includes all active and inactive user relationships plus user details. This is useful for:

  • Backing up your complete follow history (active and inactive users)
  • Transferring data between browsers or computers
  • Restoring data after reinstalling the extension
  • Preserving historical relationship data for analysis

About Username Updates

Usernames are now updated automatically! Whenever you open the extension and view your followers/following list, any missing or outdated usernames will be automatically updated.

The button below is provided for manual updates if needed, but in most cases, you don't need to use it anymore.

This optimization reduces the number of API calls made to Duolingo servers.
